# Sales Script

## How To Position It

Do not sell this as:

- React website
- Vite app
- admin panel
- modern frontend

Sell it as:

```text
property lead generation website with WhatsApp and admin control
```

## Simple Desi Pitch

Use this:

```text
Bhai ye normal brochure website nahi hai.
Isme aapke buy/rent/sell pages alag hain, WhatsApp lead flow hai, Google Sheet me leads aati hain, aur aap khud admin se property aur content update kar sakte ho.
Matlab website sirf dekhne ke liye nahi, lead lane ke liye bani hai.
```

## Fast WhatsApp Pitch

```text
Sir ek premium property website setup hai Gurgaon dealers ke liye.

Isme:
- WhatsApp lead flow
- Google Sheet me automatic lead capture
- admin se khud listing update
- buy / rent / sell pages
- locality pages
- mobile-friendly design

Aapko baar baar developer ke paas jaane ki zarurat nahi padegi basic updates ke liye.

Agar chaho to demo dikha deta hoon.
```

## Who To Sell To

Best buyer:

- Gurgaon property dealer
- luxury broker
- boutique advisory firm
- broker with 2 to 10 people
- someone already running WhatsApp-based sales

Bad buyer:

- broker who only works through raw referrals
- person who wants a Rs 2,000 brochure site
- hyper-price-sensitive client wanting unlimited edits

## What To Say They Get

- premium looking website
- WhatsApp-first conversion
- Google Sheets lead storage
- hidden admin panel
- listing control
- locality content control
- contact page with real map
- no full backend headache

## Best Practical Price Positioning

### Low-end close

```text
Rs 25,000
```

Use this only if:

- you want quick close
- you want first few clients
- scope is limited

### Better package

```text
Rs 35,000 to Rs 50,000
```

Include:

- website setup
- Gurgaon content setup
- map + images + branding setup
- lead form + WhatsApp setup
- 15 to 30 days support

## Scope Boundary

At lower pricing, clearly say:

- 1 city only
- 10 to 15 listings setup
- 3 to 5 locality pages
- limited revisions
- hosting/domain extra if needed

## Practical Objection Handling

### Objection: "Website se lead aayegi kya?"

```text
Sir website akeli magic nahi karegi, but WhatsApp aur proper presentation ke saath credibility aur direct enquiry dono improve hote hain.
Abhi zyada tar dealers random portal jaisa lagte hain. Ye aapko alag dikhata hai.
```

### Objection: "Humko khud change karna hai."

```text
Admin diya hua hai. Aap khud properties, localities, FAQs aur basic settings update kar sakte ho.
Har chhoti cheez ke liye developer ko call nahi karna padega.
```

### Objection: "Itna mehenga kyu?"

```text
Sir ye sirf 4-page static website nahi hai.
Isme lead capture, WhatsApp flow, Google Sheet backend, admin editing aur proper property structure already built hai.
```

## Best Demo Flow

When showing the site:

1. homepage
2. buy page
3. one property detail page
4. contact page
5. admin panel
6. show how property edit works

## Best Closing Line

```text
Aapko ek aisi website milegi jo sirf dekhne ke liye nahi, actual enquiry aur follow-up ke liye bani hai.
```
